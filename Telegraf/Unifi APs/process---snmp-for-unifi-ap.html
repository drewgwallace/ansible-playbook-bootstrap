Process - SNMP for Unifi AP

Assumption: Telegraf is already intalled and working

Â 

In the Unifi wireless controller enable SNMP in settings, make note of the community string

Run these commands on the server monitoring SNMP

sudo apt-get update
sudo apt-get install snmp snmp-mibs-downloader
sudo download-mib

Additional MIB libraries are needed, place them in "/usr/share/snmp/mibs", wget the following links:


https://github.com/pgmillon/observium/blob/master/mibs/FROGFOOT-RESOURCES-MIB
